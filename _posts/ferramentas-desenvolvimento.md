## Desenvolvimento Frontend

Ferramentas e bibliotecas para construir interfaces web.

### Ferramentas CSS

- [Autoprefixer.github.io](https://autoprefixer.github.io/): Para converter CSS para suportar todos os browsers.
- [Codepen.io/stephenpaton-tech/full/JjRvGmY](https://codepen.io/stephenpaton-tech/full/JjRvGmY): Customizador de scrollbar.
- [Css-Tricks](https://css-tricks.com/): Diversas funcionalidades e ideias de CSS.
- [CSSGrid Generator](https://cssgrid-generator.netlify.app/): Gerador de grid.
- [UiVerse](https://uiverse.io/all): Animações de hover HTML/CSS.
- [Caniuse.com](https://caniuse.com/): Verificar compatibilidade de funcionalidades em navegadores.

### Bibliotecas Frontend

- [Embla-carousel.com/examples/predefined](https://www.embla-carousel.com/examples/predefined/): Carrosséis.
- [Tiptap.dev](https://tiptap.dev/): Editor de texto.
- [Editorjs.io](https://editorjs.io/): Editor de texto de mais alto nível.
- [Radix-ui.com](https://www.radix-ui.com/): Componentes não estilizados.
- [React-pdf.org](https://react-pdf.org/): Gerar PDF no frontend.
- [Squoosh](https://squoosh.app/): Compressor de imagens para frontend.
- [Pqina.nl/pintura/](https://pqina.nl/pintura/): Editor de imagem JavaScript.
- [Stepperize](https://stepperize.vercel.app/): A lightweight, flexible and type-safe library for creating intuitive step-by-step experiences with minimal boilerplate and maximum control.
